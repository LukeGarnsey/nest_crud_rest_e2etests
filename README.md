## Description

Nestjs test backend application with RESTapi, CRED, and E2E testing.
Setup to run locally with Docker.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod

# setup prisma & Docker
$ yarn db:dev:restart
```

## Test

```bash
# e2e tests
$ yarn test:e2e
```
